Web works but phone spins almost always means the phone cannot reach your PC over the network (common with Ethernet PC + Wi‑Fi phone), not that you used the wrong command.

Step 1 — stop the current server
In CMD press Ctrl + C.

Step 2 — start with tunnel (best fix):


cd /d D:\LiMVPapp\PingShuiPaymnent
npx expo start --tunnel -c

OR

npm run start:tunnel


Step 3 — on phone
Expo Go → Scan QR code → scan the new QR from tunnel mode.

Wait 1–3 minutes on first load (tunnel is slower than web).

If tunnel is slow / fails
